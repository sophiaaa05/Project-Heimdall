# Project-Heimdall
archaeological-gis-prospecting

**Status:** 🏗️ `IN PROGRESS` (Data Acquisition & Calibration Phase)

> *"In Norse mythology, Heimdall is the watchman of the gods who can see for a hundred leagues. This project applies that 'divine' sight through high-resolution LiDAR to uncover hidden history beneath the Swedish canopy."*

---

## 🛰️ Project Overview
This project represents a fusion of **Communication Systems Engineering** and **Landscape Archaeology**. By utilizing aerial **LiDAR (Light Detection and Ranging)** data, I am developing a workflow to "strip away" modern vegetation and infrastructure to reveal anthropogenic subsurface features specifically Viking-age burial mounds, stone circles, and ancient settlement foundations.

### 🎯 Objective
To build a reproducible data pipeline in **QGIS** and **Python** that transforms raw elevation data into high-contrast relief models, enabling the identification of archaeological sites that are invisible to the naked eye.

---

## 🛠️ Tech Stack & Methodology

| Component | Technology | Application |
| :--- | :--- | :--- |
| **Data Source** | **Lantmäteriet Laserdata NH** | High-precision Swedish LiDAR datasets. |
| **GIS Engine** | **QGIS 3.x** | Spatial analysis, layer management, and cartography. |
| **Signal Processing** | **RVT (Relief Visualization Toolbox)** | Generating Sky-View Factor (SVF) and Local Relief Models (LRM). |
| **Data Science** | **Python (GeoPandas / Rasterio)** | Automating coordinate transformations and raster clipping. |
| **Verification** | **RAÄ Fornsök Database** | Cross-referencing findings with the Swedish National Heritage Board. |
